### 데이터베이스 접속 문제

- ch11-jpa-shop 과 ch12-springdata-shop 은 웹 애플리케이션입니다. 별도의 JSP / Servlet 엔진을 쓰지 않고 간편하게 메이븐 jetty
플러그인을 쓰시면 됩니다. jetty:run 골(goal)을 실행시키신 다음 브라우저에서 http://localhost:8080/
을 접근하면 웹 시스템을 보시고 테스트하실 수 있습니다.
